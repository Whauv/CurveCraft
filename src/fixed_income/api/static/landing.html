CurveCraft

Production-Style Fixed Income Pricing and Risk Engine

CurveCraft gives you bond pricing, yield curve bootstrapping, key-rate risk analytics, and portfolio DV01 hedging in one workflow. It is designed for learning, rapid prototyping, and practical rates analytics.

Open Product View GitHub Repo
Python 3.10+ FastAPI + Plotly Pricing + Curves + Risk

What You Can Do

  • Price fixed-rate and zero-coupon bonds
  • Bootstrap spot curves from deposit and swap quotes
  • Compute duration, convexity, DV01, and key-rate DV01
  • Run portfolio scenario analysis and hedge sizing

Install

Set up the project locally in a few commands:

python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
pip install -e .

Run

Launch the API and dashboard:

python -m uvicorn fixed_income.api.main:app --reload --app-dir src

Then visit:

Landing:  http://127.0.0.1:8000/landing
Product:  http://127.0.0.1:8000/?page=bond
API Docs: http://127.0.0.1:8000/docs

Core Modules

  • bonds: cash-flow generation, clean/dirty pricing, YTM
  • curves: bootstrapper, interpolators, zero-curve math
  • analytics: duration, convexity, DV01, key-rate DV01
  • portfolio: aggregate risk and hedge notionals

Notes

  • Rates are decimals internally (5% = 0.05).
  • Day counts supported: ACT/ACT, 30/360, ACT/360.
  • Business-day schedule adjustment: Modified Following (weekday-only).
